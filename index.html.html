HTML:

HTML Stands for hyper text markup language. It is used to create the structure for the website.

It acts as a skelton. It is used to create webpages, it consists of series of elements which is represented by tags

CSS:

CSS Stands for Cascading Style Sheets is a style sheet language used with HTML or XML documents.

It allows for the separation of content from presentation, influencing layout, colors, and fonts.

CSS operates on a rule-based system: styles are determined by rules that consist of selectors and declarations.

This separation promotes content accessibility and streamlines the structure of documents.

JAVA SCRIPT:

JavaScript is a scripting or programming language that enables you to create dynamically updating content, control multimedia, animate images, and much more on web pages

It is a single-threaded, synchronous language widely used in web development, playing a crucial role in both front-end and back-end development

JavaScript is interpreted through web browsers, making it highly accessible and versatile

REACT.JS:

React, also known as ReactJS, is a popular and powerful JavaScript library used for building dynamic and interactive user interfaces, primarily for single-page applications (SPAs).

It was developed and maintained by Facebook and has gained significant popularity due to its efficient rendering techniques, reusable components, and active community support.

BACKEND

NODE.JS:

Node.js is an open-source and cross-platform JavaScript runtime environment that allows developers to execute JavaScript code outside of a web browser.

It was released in 2009 by Ryan Dahl and has since become a popular tool for building server-side applications

PYTHON:

Python is a high-level, interpreted, and object-oriented programming language known for its simplicity and readability.

It is Created by Guido van Rossum and first released in 1991, Python has become one of the most popular programming languages due to its versatility and ease of use.

Python Key Features:

Python's syntax is designed to be intuitive and easy to learn, making it an excellent choice for beginners

It supports multiple programming paradigms, including procedural, object-oriented, and functional programming

Python's dynamic typing and dynamic binding capabilities make it suitable for Rapid Application Development

JAVA:

Java is a high-level, class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible. It is a general-purpose programming language intended to let programmers write once, run anywhere,meaning that compiled Java code can run on all platforms that support Java without the need to recompile.

Java applications are typically compiled to bytecode that can run on any Java virtual machine (JVM) regardless of the underlying computer architecture. The syntax of Java is similar to C and C++, but has fewer low-level facilities than either of them. The Java runtime provides dynamic capabilities (such as reflection and runtime code modification) that are typically not available in traditional compiled languages.

SQL:

SQL stands for Structured Query Language, a language designed to manage and manipulate databases.

It allows users to:

1.Execute queries

2.Retrieve data

3.Insert, update, and delete records

4.Create new databases and tables

My SQL:

MySQL is a widely-used open-source Relational Database Management System (RDBMS) that utilizes Structured Query Language (SQL) for database operations. Developed by MySQL AB and now owned by Oracle Corporation, MySQL is known for its reliability, speed, and ease of use. It is a key component in many web applications, forming the backbone of popular websites and services.
